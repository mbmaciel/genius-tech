
# Genius Tech

Robo de operações binárias.

## Como rodar

Antes é preciso instalar a versão 10.19.0 do node *:

```
nvm install 10.19.0
nvm use 10.19.0
```

```
git clone https://github.com/mbmaciel/geniustechnology.git
cd geniustechnology
npm install
npm start
```


## Atualizar para ultima versão

```
git pull 
npm install
```

* não funciona com outras versões.
